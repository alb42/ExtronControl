ExtronControl
-------------

Extron DSC 301 HD control program for Amiga68k and MorphOS.

very first alpha release.
currently device name is hardcoded
Amiga68k: serial.device (device is connected via serial port at backside)
MorphOS: usbmodem.device (device is connected via USB at from panel)

currently supported:
- Switch Input
- Auto Input Switch
- Auto align picture
- Test images

When the program starts it shows the Firmware version in the statusbar
it is tested with 1.14 and 1.25 (last number)

Source included, GUI created used MUIIDE and MUIClass
